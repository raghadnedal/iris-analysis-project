# Iris Dataset Analysis

This project analyzes the famous Iris dataset using Python.

The goal of this project is to practice data analysis and visualization using:

- Pandas
- NumPy
- Matplotlib
- Git and GitHub

## Project Description

In this project, I explored the Iris dataset, which contains measurements of different Iris flower species.

The dataset includes:

- Sepal length
- Sepal width
- Petal length
- Petal width
- Species

## What I Did

- Loaded the dataset using Pandas
- Explored the data using `head()`, `info()`, `describe()`, and `shape`
- Checked for missing values
- Analyzed flower species using `groupby` and `pivot_table`
- Created new features such as:
  - `sepal_area`
  - `petal_area`
- Used NumPy for:
  - mean, min, max, and standard deviation
  - normalization
  - standardization
  - distance calculation between flowers
- Created visualizations using Matplotlib:
  - Scatter plot
  - Histogram
  - Bar chart
  - Pie chart

## Main Insights

- The dataset contains three Iris species.
- The dataset is balanced, with the same number of samples for each species.
- Petal length and petal width are useful features for distinguishing between species.
- NumPy distance calculations are related to the idea of K-Nearest Neighbors in Machine Learning.
